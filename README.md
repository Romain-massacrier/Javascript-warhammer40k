# Javascript-warhammer40k

Boutique immersive inspirée de l’univers Warhammer 40k, développée en JavaScript avec une interface Adeptus Mechanicus et un système de réquisition d’objets.

## Aperçu du projet

Ce projet met en scène une boutique interactive où l’utilisateur peut acheter des reliques de guerre issues de l’Imperium.

Le thème visuel est basé sur l’Adeptus Mechanicus :

* ambiance terminal de forge
* interface sombre et industrielle
* vocabulaire inspiré du lore Warhammer 40k

## Fonctionnalités

* Affichage dynamique des objets
* Système d’achat avec gestion des crédits
* Gestion du stock en temps réel
* Filtre par catégorie :

  * Armes
  * Armures
  * Potions
  * Artefacts
* Interface immersive type terminal Mechanicus
* Projet 100% JavaScript côté client

## Technologies utilisées

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Manipulation du DOM
* Git & GitHub

## Univers

Le projet s’inspire de l’Adeptus Mechanicus, faction emblématique de Warhammer 40k :

> "La donnée est sacrée. La machine est éternelle."

Chaque objet de la boutique représente une relique, un équipement militaire ou un artefact technologique récupéré sur un champ de bataille.

## Convention de commits

Exemples utilisés dans ce projet :

```
init: création du projet
feat: ajout des objets warhammer
feat: ajout système d'achat
style: thème adeptus mechanicus
fix: correction bug stock
refactor: amélioration du rendu
docs: ajout README
```
